# SE Copilot — Context Changelog

This file tracks all meaningful updates to `context/PRODUCT_CONTEXT.md`.
Update it every time you push a change so teammates know what improved and why.

---

## How to use this file

When you update the context file, add an entry here **before** pushing:

```
## [version] — YYYY-MM-DD
**Updated by:** [Your name]
**Sections changed:** [e.g., Section 4, Section 6]

### Added
- [What new content was added and why]

### Changed
- [What was modified and what triggered the change]

### Removed
- [What was removed and why it was outdated]

### Deals that informed this update
- [Won/Lost deal, company type, key learning]
```
